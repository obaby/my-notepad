File

  • Process 
    Use this option to launch the actual processing of the program 
wich should be loaded in the filename list box first.
  • Open Project 
    Opens a already saved project.
  • Save Project 
    Saves a project.
  • Save Project As 
    Saves a project with another name.
  • Load Symbol File 
    Loads a symbol file.
Look in Hints and Tips to find out what a symbol file is.
  • Exit 
    No comment.

Dumpers

  • DSF Builder 
    Builds DSF files.
Look in Hints and Tips what DSF files are.
  • DCU Dumper 
    With this option you can dump any DCU file without loading a process.


Tools

  • PE Editor 
    With this tool you can edit various option of a PE file.
  • Dump Active Process 
    With this option you can dump (process) a packed exe directly from memory.
(Note from xavier : This is a great tool.)
  • DOI Builder 
    Builds DOI files, look in Hints and Tips to find out what DOI files are.
  • RVA Converter 
    Converts offset to physical location.
  • Opcode to Assembler 
    Dumps a given opcode to his assembler form.


Options

  • Symbols 
    Symbols dialog box, look in Hints and Tips 
and Steb by Step to find out more.
  • Configuration 
    Configuration dialog box, look in Hints and Tips 
and Steb by Step to find out more.
